Pearo Documentation

Introduction

Pearo is a beautifully crafted new modern HTML template for the Insurance Company and Agency website. The template is created with Bootstrap 4.3.x and CSS. Also, Sass is included in the template bundle. Retina ready homepage and individual pages design will best fit to use this template for all types insurance company and agency website such as home insurance, car insurance, health insurance, life insurance, travel insurance, business insurance, social insurance, fire insurance, marine insurance, property insurance, liability insurance, and many more.

Pearo is developer-friendly, fast, lightweight, clean and contains W3 valid source code. The template is easily customizable and expandable with details documentation. Our fast and friendly support team will glad to assist with any needs with the template.

Template Features:

Getting Started

Unzip the archive and find out the folder pearo or pearo-rtl There are all the template files in this folder.

The file and folder structure is something like this:

  1. assets/css — folder with css files.
  2. assets/fonts — folder with fonts files.
  3. assets/img — folder with image files.
  4. assets/js — folder with Javascript files.
  5. assets/php — folder with form-process.php files.

Upload the template files to server with the help of one of the FTP-clients like FileZilla.

Fonts used

By default, the template loads Open Sans and Roboto font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.css" file:


@import url('https://fonts.googleapis.com/css?family=Open+Sans:300,300i,400,400i,600,600i,700,700i,800,800i&display=swap');
@import url('https://fonts.googleapis.com/css?family=Roboto:100,100i,300,300i,400,400i,500,500i,700,700i,900,900i&display=swap');
